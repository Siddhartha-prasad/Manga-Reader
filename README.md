# 📚 Manga Reader

A modern **manga reading web app** built with React and Node.js that allows users to search manga, view chapters, and read pages online.

This project uses the **MangaDex API** to fetch manga data and chapter images.

# ✨ Features

* 🔎 Search manga by title
* 📖 View manga chapters
* 🖼️ Read manga pages in a vertical reader
* 📚 Recommended manga on homepage
* 🌙 Dark themed UI
* 📱 Responsive layout
* ⚡ Fast image loading
* ⬅➡ Chapter navigation
